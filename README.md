리액트 연습(포켓몬 카드 검색 페이지) : React, pokemontcg api

모든 페이지에는 haeder(로고, 검색창). 카드를 보여주는 content, footer(Pokémon TCG API를 사용하여 데이터를 제공하는 웹사이트임과 저작권 및 상표권에 대한 설명) 으로 구성되어 있습니다.
